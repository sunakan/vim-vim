== Vimメモ

=== バッファ

`:ls` or `:buffers` でバッファ一覧

`%` はカレントバッファ

`#` は代替バッファ

`:bw hoge` はバッファの削除

=== ウィンドウ(ペインみたいなもの)

`:new` 空のウィンドウを水平分割で開く

`:vnew` 空のウィンドウを垂直分割で開く


=== タブ

`:tabnew {file}` {file}を新しいタブページで開く

`:tabmove [N]` カレントタブページをN番目の後に移動
`:tabmove+[N]` カレントタブページをNだけ後に移動
`:tabmove-[N]` カレントタブページをNだけ前に移動

`gt` 次のタブに切り替え
`gT` 前のタブに切り替え

=== カレントファイルの実行

`:w !bash`

=== セッションをファイルに保存して復元

`:mksession {filename}`

`:source {filename}`

.起動時に読み込む
----
$ vim -S {filename}
----
